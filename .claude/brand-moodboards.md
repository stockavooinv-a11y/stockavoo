```html
  text-decoration: none;

<!-- Premium Badge (Option 3: Aubergine) -->
<div style="
  background: linear-gradient(135deg, #7C3AED 0%, #D4A574 100%);
  color: #FFFFFF;
  padding: 8px 16px;
  border-radius: 20px;
">Premium</div>
```

### Dashboard Components
- **Sidebar**: Use primary color as background
- **Cards**: Soft background colors (Lavender, Lilac, Pearl)
- **Charts**: Mix of primary, secondary, and accent colors
- **Buttons**: Primary gradient with white text

### Design Principles
1. **Hierarchy**: Primary purple for important actions
2. **Contrast**: Always test text readability (WCAG AA minimum)
3. **Consistency**: Use same gradient angle (135deg) throughout
4. **Balance**: 60% neutral, 30% primary, 10% accent
5. **Accessibility**: Minimum 4.5:1 contrast ratio for text
